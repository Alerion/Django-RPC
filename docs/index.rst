Django RPC documentation
========================

Django RPC for jQuery. Allows execute some server-side functions with Javascript in your browser.

Inspired by Ext.Direct.


Support
-------

You can ask question or report bug on `project Github repo <https://github.com/Alerion/Django-RPC/issues>`_.


Requirements
------------

Django RPC requires the following:

* Python (2.6.5+, 2.7)
* Django (1.3, 1.4, 1.5)
